## What I do

- Пишу GitHub Actions/GitLab CI конфигурации

- Организую staging и production деплой
- Создаю проверку качества кода (lint, test, build)
- Настрою автоматическую публикацию пакетов

## When to use me

Используй меня, когда нужно:

- Создать CI/CD пайплайн для проекта
- Добавить автоматическое тестирование
- Организовать deploy на staging
- Настроить автоматический release
