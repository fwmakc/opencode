## What I do

- Пишу тесты с Vitest API (describe, it, expect)
- Использую chai assertion library
- Работаю с mocking через vi.mock
- Организую test suites и fixtures
- Создаю coverage отчеты

## When to use me

Используй меня, когда нужно:

- Создать unit тесты для Vite проектов
- Использовать native ESM поддержку
- Настроить быстрое тестовое окружение
- Создать тесты для React/Vue компонентов
