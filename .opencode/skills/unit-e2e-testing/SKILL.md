## What I do

- Пишу unit тесты для функций и классов
- Создаю mock/stub для внешних зависимостей
- Организую E2E тесты для полного flow
- Использую best practices для тестируемого кода
- Создаю assertions и ожидания

## When to use me

Используй меня, когда нужно:

- Создать тесты для нового кода
- Добавить покрытие для legacy кода
- Организовать непрерывную интеграцию
- Проверить корректность изменений
