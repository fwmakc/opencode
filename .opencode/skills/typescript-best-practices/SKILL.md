## What I do

- Пишу строго типизированный TypeScript код
- Использую advanced types (generics, mapped types, conditional types)
- Создаю module declarations и ambient types
- Организую проекты с tsconfig.json
- Пишу type-safe API и interfaces

## When to use me

Используй меня, когда нужно:

- Создать типы для новых данных
- Организовать типизацию legacy кода
- Создать type-safe библиотеку
- Настроить строгую проверку типов
