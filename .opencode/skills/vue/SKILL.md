## What I do

- Пишу компоненты Vue 3 с Composition API
- Создаю реактивные состояния через ref, reactive, computed, watch
- Работаю с Vue Router для навигации
- Управляю состоянием через Pinia
- Пишу переиспользуемые компоненты с правильной инкапсуляцией
- Работаю с Vue Directives и Hooks

## When to use me

Используй меня, когда нужно:

- Создать новый Vue компонент
- Реализовать реактивное состояние
- Настроить навигацию в приложении
- Интегрировать third-party Vue плагины
