## What I do

- Пишу тесты с Jest API (describe, it, expect)
- Использую mock/spy/stub функции
- Работаю с snapshot testing
- Создаю custom matchers
- Организую test suites и fixtures

## When to use me

Используй меня, когда нужно:

- Создать unit тесты для JS/TS кода
- Настроить тестовое окружение
- Создать snapshot тесты
- Использовать изолированный тестовый контекст
