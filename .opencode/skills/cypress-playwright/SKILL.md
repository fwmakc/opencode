## What I do

- Пишу E2E тесты с Cypress и Playwright API
- Работаю с постраничными тестами и шагами
- Создаю fixture и mock данные
- Организую parallel тестирование
- Создаю отчеты и видео recordings

## When to use me

Используй меня, когда нужно:

- Создать E2E тесты для веб-приложения
- Проверить UI flow и пользовательский опыт
- Организовать regression тестирование
- Создать тесты для headless и browser запуска
