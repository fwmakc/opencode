## What I do

- Пишу функциональные React компоненты с хуками
- Создаю управляемые формы и обработку событий
- Работаю с React Router для навигации
- Управляю состоянием через Context API и Redux
- Оптимизирую производительность через memo, useMemo, useCallback
- Работаю с React Suspense и lazy loading

## When to use me

Используй меня, когда нужно:

- Создать новый React компонент
- Реализовать сложное пользовательское состояние
- Настроить навигацию в приложении
- Оптимизировать производительность React приложения
