## What I do

- Разделяю код на слои (Entities, Use Cases, Interface Adapters, Frameworks)
- Создаю зависимости от абстракций, не деталей
- Пишу бизнес-логику независимой от фреймворков
- Работаю с dependency inversion через 接口/абстракции
- Создаю слой presentation без зависимостей от domain

## When to use me

Используй меня, когда нужно:

- Создать масштабируемую архитектуру приложения
- Разделить бизнес-логику от инфраструктуры
- Сделать код тестируемым без framework зависимостей
- Организовать проект для работы большой команды
