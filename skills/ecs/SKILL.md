## What I do

- Создаю ECS архитектуру с Entitas,specs или other Rust/TS библиотеками
- Пишу компоненты данных без логики
- Создаю системы для обработки компонентов
- Оптимизирую memory layout и cache locality
- Реализую event-driven communication между системами
- Работаю с delta time и fixed time steps

## When to use me

Используй меня, когда нужно:

- Создать игровую архитектуру на ECS
- Организовать high-performance игровой цикл
- Добавить новую игровую механику через компоненты
- Оптимизировать производительность через data-oriented design
